Write a Python function that downloads content from a specified S3 bucket, converts the response to JSON, and handles any request exceptions.

You can use the 'boto3' library in Python to interact with Amazon S3. Here's a function that downloads a file from a specified bucket, transforms the response to JSON, and handles exceptions during the process.

```python
import boto3
import json

def download_file_from_s3(bucket_name, key, file_name):
    s3 = boto3.client('s3')
    try:
        s3.download_file(bucket_name, key, file_name)
    except ClientError as e:
        print(f"Error downloading file from S3 bucket: {e}")
    else:
        with open(file_name) as f:
            return json.load(f)
```

This function takes three parameters:
- `bucket_name`: The name of the S3 bucket from which you want to download a file.
- `key`: The key of the file in the S3 bucket.
- `file_name`: The local file name to store the downloaded file.

It creates an S3 client and tries to download the file using the specified bucket and key. If there's an error, it prints the error message. If the download is successful, it opens the file and returns its content as a JSON object.
